A Telegram bot for connecting passengers and drivers on the Namangan-Tashkent route.

## Features
- **Passengers**: Order taxi, send parcels, view driver details.
- **Drivers**: Bid on orders, set active routes, manage profile.
- **Admin**: Approve/Reject driver applications.

## Tech Stack
- Node.js
- Grammy.js (Telegram Bot Framework)
- MongoDB (Mongoose)
- Docker support

## Setup
1. Clone repository
2. `npm install`
3. Create `.env` file with credentials
4. `npm start`
